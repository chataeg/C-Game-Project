# C-Game-Project

2021-1 학기 SW설계기초의 4인 게임 프로젝트.


C언어의 기본적인 기능만 사용해 제작했다. 게임은 충돌 처리를 통해 캐릭터가 장애물을 피하며 조건을 충족해 목적지에 도달하는 방식이다.
